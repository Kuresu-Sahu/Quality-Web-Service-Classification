# Semi-Supervised Learning for Quality-Based Web Service Classification using CatBoost

This project presents a machine learning-based approach for classifying web services according to their Quality of Service (QoS) attributes using the powerful CatBoost algorithm combined with semi-supervised learning techniques. The system is designed to improve the accuracy and efficiency of web service classification when labeled training data is limited.

The proposed model analyzes various QoS parameters such as response time, availability, reliability, throughput, and latency to categorize web services into different quality levels. By leveraging semi-supervised learning, the project utilizes both labeled and unlabeled datasets, reducing the dependency on large amounts of manually labeled data.

## Key Features

* Semi-supervised machine learning approach
* Quality-based web service classification
* CatBoost classifier for high accuracy and efficient training
* Data preprocessing and feature engineering
* Performance evaluation using standard metrics
* User-friendly web interface using HTML, CSS, and JavaScript
* Fast and scalable prediction system
